# AGENTS.md — инструкции для Codex

Проект: статический сайт гостевых домов «Маривилл».

## Главные правила

1. Не ломать русские имена папок и файлов.
2. Не переименовывать существующие папки в `assets`, если это не требуется явно.
3. При изменении галерей проверять, что все пути к фото реально существуют.
4. Сайт статический: основной файл — `index.html`.
5. CSS и JavaScript сейчас находятся внутри `index.html`.
6. Не добавлять сборщики, React/Vite/Next.js без явного запроса.
7. Перед крупными изменениями сохранять рабочую версию.
8. После изменений проверять:
   - открытие фото домов;
   - галерею «Небольшая подборка фото Маривилл»;
   - блок «Баня и фурако»;
   - блок «Дополнительные услуги»;
   - мобильную шапку;
   - карту и ссылку на Яндекс.Карты.

## Кодировка и архивы

В проекте много русских имён файлов. При распаковке и упаковке использовать системные команды:

```bash
unzip archive.zip -d folder
zip -qr output.zip .
```

Не использовать инструменты, которые могут превратить русские имена папок в нечитаемые символы.

## Стиль сайта

Общий стиль:
- природный;
- тёплый;
- аккуратный;
- не слишком «люкс», но визуально приятный;
- зелёный, кремовый, золотистый акценты.

## Контент

Не использовать формулировку «база отдыха». Использовать:
- «гостевые дома»;
- «Маривилл»;
- «гостевые дома у Волги».
