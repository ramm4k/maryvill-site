Статический сайт гостевых домов «Маривилл» у Волги.

```text
Maryvill/

```

## Как открыть локально

Самый простой способ:

1. Открыть папку проекта.
2. Дважды нажать `index.html`.
3. Проверить сайт в браузере.

Для более корректной проверки лучше запустить локальный сервер:

```bash
python3 -m http.server 8000
```

Затем открыть:

```text
http://localhost:8000
```

## Публикация через GitHub Pages

1. Создать репозиторий на GitHub.
2. Загрузить содержимое папки `Maryvill`, а не сам ZIP-архив.
3. Включить GitHub Pages:
   - Settings → Pages
   - Source: Deploy from branch
   - Branch: main
   - Folder: /root

## Важно

- Не переименовывать папки в `assets`, потому что пути к фото прописаны в `index.html`.
- При упаковке архива использовать системный `zip`, чтобы не ломались русские названия папок.
- Для новых фото лучше добавлять отдельную папку в `assets` и обновлять массивы галерей в `index.html`.
